# Knowledge Rotation System

Enterprise knowledge rotation system designed to identify operational risks
caused by employee turnover or temporary absences.

## Purpose
Help companies detect areas that rely heavily on a small number of key people
and proactively plan knowledge transfer strategies.

## Core Features
- Employee management
- Area / department management
- Knowledge registration
- Knowledge level per employee
- Area risk detection based on knowledge dependency

## Tech Stack
- Backend: Node.js + Express
- Frontend: React
- Database: PostgreSQL
- Version Control: Git & GitHub
